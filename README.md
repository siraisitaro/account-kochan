⛪ Gospel Quest: Noah's Logic (ゴスペルクエスト：ノアの論理)

「主の平和は、まず論理的な整合性から」

北東北の寒冷都市「森岡シティ」を舞台に、福音派の優男・ノア牧師が聖書的ロジックを武器に異端やサタンと戦う、前代未聞のブロック崩しRPGです。

🌟 ストーリー

岩手山の麓、森岡シティの片隅にある「みなみチャーチ」。
ここに赴任した長身眼鏡のノア牧師を待ち受けていたのは、街を蝕む不気味な異端の影だった。

ノアは眼鏡をクイッと押し上げ、静かに告げる。
「あなたの解釈は文脈を無視しています。……Q.E.D.（証明終了）」

今、聖書を携えたロジカルな救済の戦いが幕を開ける！

🎮 ゲームの特徴

アドベンチャーパート: 森岡シティで巻き起こる様々な事件を追う本格ストーリー。

ロジカル・ブレイク（バトル）: 敵の「詭弁」という名のブロックを、「福音の弾丸」で打ち砕くブロック崩しアクション。

神学的深み: 福音派の教義に基づいた（？）熱い論戦演出。

👓 キャラクター紹介

ノア牧師

みなみチャーチの新任牧師。眼鏡をかけた長身の優男。
非常にロジカルな思考の持ち主で、どんな敵の誘惑も神学的根拠に基づいて完璧に論破する。
口癖は「聖書をよく読んでください」。
